# File Guide

## 1. `sequence_templates_12.csv`

Purpose:

- defines the 12 sequence templates
- shows the exact gloss order for each `SEQ`

Use by:

- recording group
- review group
- annotation group

---

## 2. `sequence_recording_manifest_300.csv`

Purpose:

- master list for all 300 sequence videos
- tells who records which file and how many takes are expected

Use by:

- recording group
- review group
- project lead

---

## 3. `sequence_review_checklist_300.csv`

Purpose:

- review checklist for each recorded video
- used to decide whether a clip is accepted or must be re-recorded

Use by:

- review group

---

## 4. `sequence_annotations_300_template.csv`

Purpose:

- time annotation template for sequence videos
- every gloss segment is already expanded into rows

Use by:

- annotation group

---

## 5. `glossary_template.csv`

Purpose:

- standardized dictionary of all glosses used in this project

Use by:

- annotation group
- glossary/document group
- future model training pipeline

---

## 6. `gloss_description_assignment.csv`

Purpose:

- tracks which gloss descriptions need to be written

Use by:

- glossary/document group

---

## 7. `sequence_work_plan_300.md`

Purpose:

- complete project task sheet for this round
- includes team roles, templates, recording rules, and file usage

Use by:

- everyone
